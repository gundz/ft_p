#FTP
----

Build instructions: juste do "Make"

./server port
./client addr/localhost port

If you wanna test localy, just copy one of the binary to another dir


##Commands:

	remote:
	-------
	
		*get __file__
		*put __file__
		*ls
		*pwd
		*cd __path__
	
	local:
	------
	
		*lls
		*lpwd
		*lcd __path__
		*quit
